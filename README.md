💱 Currency Converter App

A **simple and beautiful currency converter** built while learning how to use **real-time APIs**.
It connects with a live exchange rate API to show accurate conversion results instantly.

# What I Learned

* How to fetch **real-time data** using APIs
* How to update content dynamically on the page
* How to design an **attractive and interactive interface**
* How to handle user input and display results smoothly

# Features

 Real-time exchange rate conversion
 Glass-style modern UI
 Country flags and easy selection
 Smooth user experience


# 🚀 How It Works

1. Enter an amount to convert.
2. Choose the currencies you want to convert from and to.
3. Click **Get Exchange Rate** — the app fetches live data using an API and shows the result instantly.
